# AnalyticsOptions

The analytics options made available in [constructCloudinaryUrl](/constructcloudinaryurl) are inherited from the Cloudinary URL Gen SDK.

Types and more information [can be found on GitHub](https://github.com/cloudinary/js-url-gen/blob/8103b542dac284d0b72ec5227d2537453c1ee53d/src/sdkAnalytics/interfaces/IAnalyticsOptions.ts).
