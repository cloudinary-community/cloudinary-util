# Installing URL Loader

## Getting Started

<Steps>

### Installation

<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tab>
    ```ansi copy
    npm install @cloudinary-util/url-loader
    ```
  </Tab>
  <Tab>
    ```ansi copy
    pnpm install @cloudinary-util/url-loader
    ```
  </Tab>
  <Tab>
    ```ansi copy
    yarn add @cloudinary-util/url-loader
    ```
  </Tab>
</Tabs>

### Import the Dependency

```jsx copy
import { constructCloudinaryUrl } from '@cloudinary-util/url-loader';
```

### Create a Cloudinary URL

```jsx copy
const url = constructCloudinaryUrl({
  options: {
    src: 'my-public-id',
    width: 800,
    height: 600
  },
  config: {
    cloud: {
      cloudName: 'my-cloud'
    }
  }
});
```
